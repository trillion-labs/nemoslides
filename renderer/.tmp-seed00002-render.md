# Campaign architecture

- Publish a joint `2026 Mid-Market Hiring Signals` report
- Run one benchmark webinar and one invite-only CHRO roundtable
- Gate every registration through a shared company-size filter
- Route sales-ready accounts to Peak within `24 hours`

# Commercial model

- Lumina funds creative, design, and paid promotion: `$22K`
- Peak funds speakers, SDR follow-up, and event ops: `$20K`
- Peak keeps deal ownership; Lumina earns a `15%` referral fee
- Continue only if pilot yields `180 MQLs`, `12 SQLs`, and `3 wins`
